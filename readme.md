# Kyverno MVP – стенд для демонстрации политик безопасности

Этот репозиторий содержит инфраструктурный код для развёртывания MVP-стенда Kyverno на одной виртуальной машине (Ubuntu/Debian). Всё управляется через Ansible, а политики синхронизируются через ArgoCD (GitOps).

## Что будет развёрнуто

- **Kubernetes** (KinD) с control-plane и двумя workers.
- **Ingress-nginx** как точка входа с пробросом портов 80/443.
- **ArgoCD** – GitOps-движок с подходом App of Apps.
- **Kyverno** – движок политик, установленный через Helm.
- **Prometheus + Grafana** – мониторинг и дашборды (включая кастомный дашборд Kyverno).
- **Policy Reporter** – веб-интерфейс для просмотра результатов политик.
- **Примеры политик** (запрет latest-тегов, привилегированных контейнеров, hostPath) и тесты к ним.

## Требования к проверяющему

- **Локальный ПК** с установленным Ansible (версия >= 2.9) и доступом по SSH к ВМ.
- **Виртуальная машина** (Ubuntu 22.04 / Debian 11+) с 8 ГБ ОЗУ, доступом в интернет и открытыми портами 80, 443, 22 (SSH).
- **Git** для клонирования репозитория.
